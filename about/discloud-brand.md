# Discloud Brand

## üß≠ Purpose

This page provides a concise, single-source reference for using the Discloud name and visual assets. Keep usage clear, consistent, and non-misleading. When in doubt, link to our platform rather than modifying brand elements.

{% hint style="warning" %}
Do not present Discloud as endorsing, sponsoring, or partnering with your project unless explicitly agreed.
{% endhint %}

***

## üß© Name & Meaning

The name combines the Portuguese / Latin-derived prefix **dis-** (divide, distribute, separate) with **cloud** (as in cloud computing).

> We are dissolving, dividing, and sharing cloud computing between people.

| Prefix | Word  | Concept                                        |
| ------ | ----- | ---------------------------------------------- |
| dis-   | cloud | Democratized access to scalable infrastructure |

***

## ‚ôæÔ∏è Symbol & Narrative

The core symbol draws from:

* A M√∂bius strip: continuous evolution and iteration.
* An abstract lidded container: balance between time and efficiency.
* Interconnected surfaces: infrastructure pieces forming a seamless platform.

These metaphors express flexibility, reliability, and accessible scale.

## üö´ Restricted / Don'ts

Do not apply transformations that dilute clarity or misrepresent the brand.

| Misuse                 | Description                   |
| ---------------------- | ----------------------------- |
| Drop shadows           | Avoid ornamental effects      |
| Outlines / strokes     | Do not add borders            |
| Stretch / skew         | Preserve aspect ratio         |
| Low resolution         | Always use proper size assets |
| Low contrast placement | Ensure sufficient contrast    |
| Recoloring             | Only use provided variants    |

If a required treatment seems outside available assets, request it!
{% endhint %}

## üîó Attribution & Linking

When referencing Discloud in articles, docs, or open-source repositories:

* Use the name "Discloud" with capital D.
* Link to [https://discloud.com](https://discloud.com/) on first mention.
* Avoid possessive forms if it creates ambiguity (prefer "Discloud platform" over informal variations).
